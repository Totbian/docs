# All Fonts

We use two main fonts across the Molin UI components:

1. **Work Sans**:

   - Regular
   - Italic
   - Bold

2. **Inter**:
   - Regular
   - Italic
   - Bold

# All Weights

We apply different font weights to enhance readability and aesthetic appeal:

- **font-light** (300): Used for secondary texts or subdued elements.
- **font-normal** (400): Default weight for most text elements.
- **font-medium** (500): Slightly emphasized text for headings.
- **font-semibold** (600): Used for section headers or important labels.
- **font-bold** (700): Used for titles and important calls to action.

# All Sizes

We utilize various text sizes for different UI components from Tailwind. Here's a breakdown of commonly used sizes:

- **text-xs**: 0.75rem
- **text-sm**: 0.875rem
- **text-base**: 1rem
- **text-lg**: 1.125rem
- **text-xl**: 1.25rem
- **text-2xl**: 1.5rem
- **text-3xl**: 1.875rem
- **text-4xl**: 2.25rem
- **text-5xl**: 3rem
- **Hero Sections**:
  - Secondary hero text-6xl
  - Main hero text-[84px]

# Landing page font classes

- Main hero: text-[52px] sm:text-[84px]

```html
text-[52px] sm:text-[84px] lg:leading-tight tracking-3 font-display leading-snug font-bold text-black
```

- H1: text-4xl sm:text-5x

```html
text-4xl lg:text-5xl leading-snug sm:leading-normal tracking-tight font-bold font-display text-black
```

- H2: text-3xl sm:text-4xl
  It can be either bold or semibold.

```html
text-3xl lg:text-4xl leading-normal tracking-tight font-semibold font-display text-black
```

- H3: text-base sm:text-lg

```html
text-base sm:text-lg font-sans font-semibold text-gray-800 tracking-tight
```

- Header subtext / paragraph: text-base, sm:text-lg
  on dark bg: text-[#9098A0]

```html
text-base sm:text-lg font-sans text-gray-500
```
